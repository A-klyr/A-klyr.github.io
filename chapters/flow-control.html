Flow control adalah cara kita mengatur alur logika program menggunakan kondisi dan perulangan.

1Ô∏è‚É£ Pernyataan Kondisional (if, elif, else)

Gunakan struktur ini untuk membuat keputusan dalam program:

if x > 0:
    print("Positif")
elif x == 0:
    print("Nol")
else:
    print("Negatif")

2Ô∏è‚É£ Perulangan (for dan while)

For loop digunakan untuk iterasi pada urutan data seperti list atau string:

for i in range(5):
    print("Nomor:", i)

While loop berjalan selama kondisi bernilai benar:

count = 0
while count < 3:
    print("Hitung:", count)
    count += 1

3Ô∏è‚É£ Break dan Continue

Untuk mengatur alur di dalam loop:

for i in range(5):
    if i == 2:
        continue  # lompat ke iterasi berikutnya
    if i == 4:
        break     # keluar dari loop
    print(i)

Output:

0
1
3
